# Homebrew Tap

Homebrew formulae and casks for [Melatonin](https://github.com/andrd3v/Melatonin) and future tools.

## Install

```bash
brew tap andrd3v/tap
brew install --cask melatonin
```

## Available Casks

| Cask | Description |
|------|-------------|
| `melatonin` | Fix overnight battery drain on macOS |
